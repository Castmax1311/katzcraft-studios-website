KatzcraftTimer 1.3

A simple but functional Timer Plugin

Native Minecraft Version: 1.19
Tested Minecraft Versions: 1.19

Description:

This plugin is very simple, but it has all of the features you need for a timer!
-You can start or resume it (/timer resume)!
-You can pause it (/timer pause)!
-You can reset it (/timer reset)!
-You can set the time (/timer time seconds)!
